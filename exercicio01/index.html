O que é HTML?

(voltar)

Html significa Hypertext markup language, ou seja significa Em português, seria algo como “Linguagem de Marcação de Hipertexto”

É a linguagem usada para estruturar páginas da internet. em HTML utilizamos coisas como:

  • Titulos
  • Parágrafos
  • Imagens
  • Listas
  • Tabelas
  • Links

Como o HTML é estruturado ?

(voltar)

HTML é estruturado através de tags e seu codigo subdivide em 3 partes sendo:

  • HTML = tudo que está dentro dele está sendo usado no site
  • HEAD = É a parte que precisa ser carregada antes do site ser ultilizado
  • BODY = Aqui é toda a parte que o usuario possa visualizar

O que são TAGS ?

(voltar)

São comandos onde cada um possui a sua função e serve para organizar o codigo para a criação de sites

São estruturados atravez de < > mas possuindo algumas caracteristicas diferentes

A maioria das tags possuiem estrutura de abertura e fechamento sendo:
( < > )abre (</>)fechamento precisa de uma barra (/) para fechar

Tags sem fechamento:
possuem somente (< >) mas na maioria das vezes usam atributos

Tags com atributos:
São complementos que adicionam informações às tags. Eles ficam dentro da tag de abertura.

Tipos de TAGS

(voltar)

Parágrafos

A tag de parágrafo é usada para escrever textos.

  • <p>Este é um parágrafo</p>

Títulos

Os títulos são representados pela letra H e vão de H1 até H6.

  • <h1>Título principal da página</h1>
  • <h2>Subtítulo</h2>
  • <h3>Subseção</h3>
  • <h4>Subdivisão</h4>
  • <h5>Título menor</h5>
  • <h6>O menor título</h6>

Citações

São usadas para indicar citações de outros autores podendo ser frases,formulas etc

É representado pela palavra blockquote

  • <blockquote> é representado por está tag </blockquote>

Negrito

São usadas para deixar palavras,frases ou textos em negrito
É representado pela tag <strong>

  • <strong> Esta frase esta em negrito </strong>

Sublinhados

sublinha uma palavra , frase ou texto
É representado pela tag <u>

  • <u> esta frase estará em negrito </u>

Italico

Transforma palavras, frases ou textos em italico
É representado pela tag <i>

  • <i> Esta frase esta em italico </i>

Marca textos

Faz uma marcaçao ao texto
É representado pela tag <mark>

  • <mark> Esta linha será marcada </mark>

Tag sup

A tag sup é usada para deixar um texto sobrescrito, ou seja, um pouco acima da linha normal do texto.

Ela é muito utilizada em expressões matemáticas ou referências.

Exemplo:

23 = 8

Listas

(voltar)

As listas são tags em HTML que ajudam na organização de estrutura do codigo
são divididas em:

Listas Ordenadas

listas que possuem marcadores numericos.
É representado pela tag <ol>

Exemplo:

  1. Primeiro item.
  2. Segundo item.

Listas Não Ordenadas

listas que possuem marcadores estilo bolinhas
É representado pela tag <ul>

exemplos:

  • Primeiro item.
  • Segundo item.
